# Proyecto de Análisis de Datos de Vehículos

Este proyecto es una aplicación web interactiva desarrollada con **Streamlit** que permite analizar un conjunto de datos de anuncios de venta de coches en los Estados Unidos.

## Funcionalidades
La aplicación proporciona herramientas visuales para explorar los datos de manera sencilla:
* **Histogramas interactivos:** Permite visualizar la distribución del millaje (odómetro) de los vehículos.
* **Gráficos de dispersión:** (Opcional si lo agregaste) Para analizar la relación entre el precio y otras variables.
* **Interfaz dinámica:** Los usuarios pueden generar los gráficos mediante botones y casillas de verificación.

## Librerias que utilice 
* Python 3
* Pandas (manipulación de datos)
* Plotly Express (gráficos interactivos)
* Streamlit (framework de la aplicación web)
